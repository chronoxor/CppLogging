Detailed Description

Logging record.

Logging record encapsulates all required fields in a single instance:

  • timestamp
  • thread Id
  • level
  • logger
  • message
  • buffer

Logging records are created inside Logger class and processed by logging appenders, filters and layouts.

Not thread-safe.
